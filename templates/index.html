Final Grade Predictor

Produced By:

Kai Velasco and Steven Zeng

Using various machine learning algorithms, our project takes several attributes from a student class performance dataset such as first semester grades, second semester grades, absences, study time, etc., and predicts what the student's final grade will be for that class. The algorithms that were employed in our project were linear regression, k-nearest neighbors, and random forest. (add more... maybe)

To understand the functionality of this project, one must first understand what machine learning is and how it works. SImply put, machine learning is the process of introducing a dataset to software, training our model on that dataset to increase accuracy, and then using this trained model to solve problems with new, unseen data. In the realm of machine learning, linear regression is a basic algorithm that evaluates scattered data points and attempts to create a best-fit line based on those data points. This best-fit line establishes a relationship between the independent and dependent variables of the problem that this algorithm is being utilized to solve. Linear regression is best implemented to evaluate data points that have strong correlation. Using this method to evaluate data points that are lacking a strong correlation will make it difficult for this algorithm to correctly identify the relationship between the points. The standard function, y = mx + b, is used to implement this algorithm where the attributes or characteristics of the data points are the input values and along with the other parts of the function, allow this algorithm to output a value which in different contexts could be something like a predicted future value. Our next algorithm, k-nearest neighbors, is a supervised learning algorithm that makes predictions based on the proximity of other data points. K-nearest neighbors takes the average of the values of the nearest neighbors and uses this to make a prediction on the value of the unknown data point. The random forest algorithm is one that outputs the decisions of multiple decision trees and uses these to come to a final decision, the average or mean of all the decision trees in the “forest”. Decision trees start with a question of some sort and based on the answer to this question, create different nodes of the tree.
